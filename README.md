# Agent-Skills

A collection of SKILL.md files I created to improve my work with AI agents. I'm sharing them hoping they help you too.

## What is This?

These are skills I developed for myself while working on UI development and other projects with AI agents. They work well for me, and I'm open-sourcing them because I believe they can help others improve their AI agent workflows.

Your workflow is different from mine, so adapt and modify these as needed. The goal is to help you get better results from your AI agent.

## How to Use Them

Reference SKILL.md files in your project when working with your AI agent. They guide the agent toward more thoughtful, production-ready decisions.

- **UI Development** — Guidance for building UIs with AI assistance
